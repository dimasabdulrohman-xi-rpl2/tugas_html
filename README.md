1. `<html>`

Fungsi:
Tag `<html>` merupakan elemen utama dalam sebuah dokumen HTML. Semua elemen atau isi dari halaman web harus berada di dalam tag ini. Tag ini menandakan bahwa dokumen yang dibuat adalah dokumen HTML sehingga dapat dibaca oleh browser dengan benar.


2. `<head>`

Fungsi:
Tag `<head>` digunakan untuk menyimpan informasi tentang halaman web yang tidak ditampilkan langsung kepada pengguna. Di dalamnya biasanya terdapat judul halaman, metadata, serta link ke file eksternal seperti CSS atau JavaScript.


3. `<title>`

Fungsi:
Tag `<title>` berfungsi untuk menentukan judul dari halaman web yang akan muncul pada tab browser. Judul ini juga penting untuk identitas halaman dan membantu pengguna mengetahui isi halaman tersebut.


4. `<body>`

Fungsi:
Tag `<body>` berisi seluruh konten yang akan ditampilkan pada halaman web, seperti teks, gambar, video, link, dan elemen lainnya. Semua yang dilihat oleh pengguna di browser berada di dalam tag ini.


5. `<h1>`

Fungsi:
Tag `<h1>` digunakan untuk membuat judul utama pada halaman web. Tag ini memiliki ukuran paling besar dibandingkan tag heading lainnya (`<h2>` sampai `<h6>`). Biasanya digunakan untuk judul penting agar mudah terlihat.


6. `<p>`

Fungsi:
Tag `<p>` digunakan untuk membuat paragraf. Paragraf berfungsi untuk menampilkan teks dalam bentuk yang lebih rapi dan mudah dibaca oleh pengguna.


7. `<a>`

Fungsi:
Tag `<a>` digunakan untuk membuat hyperlink atau tautan yang dapat mengarahkan pengguna ke halaman lain, baik di dalam website yang sama maupun ke website lain di internet.


8. `<img>`

Fungsi:
Tag `<img>` digunakan untuk menampilkan gambar pada halaman web. Dengan adanya gambar, tampilan website menjadi lebih menarik dan informatif bagi pengguna.


9. `<ul>` dan `<li>`

Fungsi:
Tag `<ul>` digunakan untuk membuat daftar tidak berurutan (unordered list), sedangkan `<li>` digunakan untuk membuat item di dalam daftar tersebut. Tag ini membantu menyusun informasi dalam bentuk daftar agar lebih terstruktur.


10. `<table>`

Fungsi:
Tag `<table>` digunakan untuk membuat tabel yang berisi data dalam bentuk baris dan kolom. Tabel biasanya digunakan untuk menampilkan data seperti jadwal, daftar nilai, atau informasi terstruktur lainnya agar mudah dipahami.
